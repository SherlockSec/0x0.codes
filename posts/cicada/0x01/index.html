Let us begin on the currently known story of Cicada 3301.

The 2012 Puzzle

The start of the first puzzle, as the initial image suggests, lies within the image itself. Opening up the image in a text editor reveals the following text at the end of the file:

TIBERIVS CLAVDIVS CAESAR says "lxxt>33m2mqkyv2gsq3q=w]O2ntk"

As the first part of that string suggests, this text is Caesar Ciphertext, specifically text that has a +4 shift applied to it.

Decyphering this text outputs an Imgur link:
